Our senior project is a **blockchain-based document approval system** built using **Hyperledger Fabric**, which ensures transparency, security, and immutability in the document approval process. The system consists of a **React frontend** for users to sign up, log in, and interact with documents, a **Node.js backend** that connects to the Fabric network through REST APIs, and **chaincode (smart contracts)** deployed on the blockchain to handle operations such as submitting, approving, or rejecting documents. Each action on a document is recorded on the ledger, providing a verifiable audit trail. The platform supports multiple organizations, ensuring trust in inter-organizational workflows like academic, governmental, or enterprise document validation.
